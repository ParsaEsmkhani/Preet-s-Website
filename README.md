# Mehndi by Preet Website

A static GitHub Pages website for Preet's henna art portfolio.

## Pages

- `index.html` - home page
- `about.html` - personal introduction
- `catalogue.html` - design catalogue
- `events.html` - event and booking types
- `news.html` - announcements
- `contact.html` - contact and inquiry form

## Update Before Publishing

Replace the placeholder contact details in `contact.html` and `script.js`:

- `preet@example.com`
- `+1 000 000 0000`
- Instagram link

Replace the starter SVG artwork in `assets/` with Preet's real henna photos when you have them.

## Publish With GitHub Pages

1. Push this folder to GitHub.
2. Open the repository on GitHub.
3. Go to Settings > Pages.
4. Choose the `main` branch and the root folder.
5. Save and wait for GitHub to publish the site.
